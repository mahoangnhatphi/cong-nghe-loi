Status

Internal sample API endpoints are active in this Flask app.

Start this app, set the environment variables below, then run examples/config.internal-api.yaml from the Run page.

Endpoints

  • GET /internal-api/old/customers/<customer_id>
  • POST /internal-api/old/orders/<order_id>
  • GET /internal-api/old/loyalty/<customer_id>
  • GET /internal-api/new/profiles/<customer_id>
  • GET /internal-api/new/order-summary/<order_id>
  • GET /internal-api/new/rewards/<customer_id>
