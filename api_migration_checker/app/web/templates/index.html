Quick Start

  1. Start the app with python3 -m api_migration_checker.app.web.main.
  2. Use the prefilled mock config path and click Validate.
  3. Click Run Check to generate SQLite, JSON, CSV, and TXT results.
  4. Open Runs to inspect summaries and mismatches.
